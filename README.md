![image](https://user-images.githubusercontent.com/75383758/127253860-d00ece3d-e3ac-42cd-aace-a3601158cfd2.png)

    a. Al cerrarse el interruptor el voltaje es **0 V** y la corriente está dada por: E/R = (20 V)/(4 Ω) = **5 A**
